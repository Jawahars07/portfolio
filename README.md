# Jawahar Naidu — Portfolio

Personal portfolio site. Built from scratch with **React + Vite + Framer Motion** — dark "Aurora / Glass" design with an animated gradient-mesh background, glassmorphism cards, and scroll-reveal motion.

**Live:** [jawaharnaidu.com](https://jawaharnaidu.com)

## Stack
- React 18 · Vite 5
- Framer Motion (animations)
- lucide-react (icons)
- Vanilla CSS design system (no Tailwind) — see `src/index.css`

## Develop
```bash
npm install
npm run dev      # local dev server
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## Deploy
Connected to **Vercel** via Git: every push to `main` auto-builds (`npm run build`) and redeploys. No manual steps.
